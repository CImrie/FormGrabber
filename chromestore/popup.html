Form Grabber

Instructions
Grabbing
  • Click 'Start Grabbing', then click all of the elements in the page to copy their data. Convenient highlighting shows you exactly which is selected.
  • Click 'Enter' to save after all have been selected, and this will be available in the list below
  • Click 'B' to go back one, i.e. if you click an element by mistake
  • Click 'C' to cancel all and stop the script. It may take 7 seconds to cancel the highlighting, but the page is ready to use immediately.
Pasting
  • Click 'use' . This will close the window and start pasting
  • Click where each element should go. The current object to be pasted is given in an alert box following your cursor
  • Once all elements have been pasted, you will receive an 'all done' message and the script will end
